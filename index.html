Y T F I N D

YTFind makes it super easy to get the YouTube transcript for videos you embed on your website. Just use our simple API to get the JSON and/or XML. We handle the hard part getting it from YouTube.



quick walkthrough sample


Sample request and response outlined below. Pass in two paramters, either xml or json output and the second being the YouTube video id. That's it. Enjoy.






                GET http://ytfind.com/json/UF8uR6Z6KLc
